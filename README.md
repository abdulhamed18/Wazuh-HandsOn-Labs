# Wazuh Hands-On Labs

This repository contains my hands-on learning journey with **Wazuh SIEM**.  
Each task focuses on a specific concept or feature of Wazuh, starting from basics and moving towards more advanced detection and response use cases.

The goal of this project is to:
- Learn Wazuh through practical labs
- Document each task clearly
- Build a personal portfolio for cybersecurity learning

---

## Learning Flow

- Task 1: What is Wazuh  
- Task 2: Lab Setup and Installation  
- Task 3: Agent Configuration  
- Task 4: Log Collection and Monitoring  
- Task 5: File Integrity Monitoring  
- Task 6: Vulnerability Detection  
- Task 7: SSH Bruteforce Detection  
- Task 8: Active Response  
- Task 9: Dashboard and Visualization  
- Task 10: Alerts and Reporting  

Each task has its own folder with a dedicated `README.md` explaining:
- Objective  
- Environment  
- Steps performed  
- Results  
- Screenshots  
- Learning outcomes
